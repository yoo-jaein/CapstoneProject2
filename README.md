## 프로젝트 소개
- 2020-2 KNU 종합설계프로젝트2 팀 bdd의 프로젝트
- 실내 COVID-19 방역 강화를 위해 마스크 미착용자를 빠르게 발견할 필요가 있다. 
- 기존의 방식은 사람이 직접 확인해야 하는 불편함이 있고, 매 순간 확인하지 못하는 문제점이 있다.
